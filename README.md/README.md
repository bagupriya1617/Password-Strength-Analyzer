# 🔐 Password Strength Analyzer in C

A console-based application written in C to analyze the strength of a password based on length, character diversity, and common security patterns.

---

## ✨ Features
- Length validation
- Uppercase and lowercase detection
- Digit and special character detection
- Repeated character identification
- Common weak pattern detection
- Strength rating with suggestions

---

## 🛠 Tools Used
- C Programming Language
- Online C Compiler (Programiz / OnlineGDB)

---

## ▶ How to Run
1. Open an online C compiler
2. Paste the code from `src/password_strength_analyzer.c`
3. Click **Run**
